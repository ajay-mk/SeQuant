struct CodeBlock

A code block groups one or multiple Results together. Upon ITF code generation a CodeBlock object will be mapped to a single ‘-&#8212; code(“xyz”)’ block in ITF. The individual results will be computed serially inside the code block and the computation of individual results will be separated in different “paragraphs” in the generated code.
