Enum Documentation

enum class sequant::mbpt::Convention

Conventions for partitioning the single-particle Hilbert space.

Values:

enumerator Minimal

occupied/hole + unoccupied/particle + their union

enumerator SR

single determinant reference: occupied (frozen + active) + unoccupied (active + frozen)

enumerator MR

multi determinant reference: occupied (frozen + active) + active + unoccupied (active + frozen)

enumerator F12

SR + complement from complete basis, used for F12 methods.

enumerator QCiFS

`Quantum Chemistry in Fock Space’ = superset of above
